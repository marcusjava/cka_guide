Dynamic provisioning Storage Classes

When a user need storage 
PV são automaticamente criadas pelos storage classes
set atributo storageClassName em PVC 


Storage Classe é um recurso de storage muito importante no kubernetes, com ele podemos provisionar persistent volumes(PV) de forma dinâmica reduzindo custos em cloud providers(ex. AWS EBS, GCE PD). 
Quando o usuario do cluster requisitar recursos de disco através PersistentVolumeClaim(PVC) o storage será criado a depender tipo de StorageClass solicitado.
